Config
------

User local logger in application.yml:
  
  > api_url: 'http://localhost:9292/v1/'

Start
-----

    > rackup

Debugging
---------

Send test notify from the browser console:

   > ICRMSendTestNotify()

Post test message to chat:

   > ICRMSendServerMessage()
